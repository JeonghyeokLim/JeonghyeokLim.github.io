Caster-Aware Control for Industrial AMRs

Jeonghyeok Lim · 2026

Overview

This project develops a caster-aware control framework for industrial autonomous mobile robots. It addresses tracking degradation caused by passive-caster misalignment, uncertain caster states, and execution-level disturbances.

Framework

  • Uncertainty-aware nonlinear model predictive control for high-level planning
  • Learning-based state estimation for caster-related latent states
  • RL-based execution layer for robust low-level tracking
  • Integrated planning and execution for industrial AMR operation
